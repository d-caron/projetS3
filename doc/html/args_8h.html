Description détaillée

Fichier de déclarations relatives aux arguments.

Déclare un nouveau type de structure permettant de stocker les arguments ainsi que toutes les fonctions permettant d'initialiser cette nouvelle structure ainsi que de récupérer, stocker, vérifier et afficher les arguments passés au programme
